Detailed Description

This executable routine is used to generate a frame index file which may be required for certain file formats.

Sassena requires trajectory files to be seekable (i.e. the frame offset position has to be known without reading the trajectory first). Since some file format do not allow to precompute the offset positions, they have to be indexed first.
